# 統一7-ELEVEn獅官方網站

這是統一7-ELEVEn獅棒球隊的非官方網站專案，提供球隊資訊、球員介紹、啦啦隊成員和售票資訊。

## 功能

- **首頁**：球隊最新消息和重要資訊
- **球員**：球員名單和個人資料
- **啦啦隊**：Uni Girls 啦啦隊成員介紹
- **售票資訊**：比賽門票購買管道和價格

## 技術

- HTML5
- CSS3
- JavaScript
- Node.js (本地開發伺服器)

## 本地開發

1. 克隆此儲存庫
2. 執行 `node server.js` 啟動本地伺服器
3. 在瀏覽器中訪問 `http://localhost:3000`

## 授權

本專案僅供學習和展示用途，非官方網站。
